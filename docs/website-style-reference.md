# Dongyuan.space Style Reference v2

## 1. Design Direction

This site should no longer feel only quiet and paper-like.
It should feel atmospheric.

The new direction is a more emotional kind of Japanese beauty:

- twilight rather than daylight
- memory rather than clarity alone
- music rather than static minimalism
- distance, weather, and light rather than plain whitespace
- cinematic tenderness rather than lifestyle neatness

This is still restrained, but no longer plain.

Working phrase:

"A luminous, rain-touched, music-shaped personal world on the web."

## 2. Emotional Target

The site should evoke:

- twilight sky after rain
- train windows and city lights
- piano resonance in a quiet room
- soft melancholy
- distance between people, but not coldness
- gentleness with emotional depth
- youth, memory, and suspended time
- luminous air and transparent color

It should feel like a place where thoughts arrive with echo and atmosphere.

## 3. Mood Keywords

Primary mood:

- atmospheric
- luminous
- wistful
- cinematic
- transparent
- musical
- tender
- reflective

Secondary mood:

- elegant
- modern
- intimate
- nightfall
- rain-softened
- emotionally spacious

Avoid:

- loud neon cyberpunk
- literal anime UI tropes
- character art as decoration
- game HUD aesthetics
- overly dark gothic styling
- overdesigned fantasy motifs
- sterile minimalist beige-only presentation

## 4. Visual Identity

### 4.1 Overall feeling

The homepage should feel like an original soundtrack turned into a website.
Not loud. Not chaotic. But full of atmosphere.

The first screen should create an emotional weather condition, not just present information.
It should feel as if light, air, and memory are present in the page.

### 4.2 Reference logic

The site should borrow from the emotional language of:

- music-driven melancholy
- Japanese animated skies and weather
- reflective coming-of-age cinema
- quiet fantasy realism
- elegant visual novels and art books

But it must not directly reference or display fandom.
The influence should be structural and emotional, not explicit.

## 5. Color System

The palette should become richer, but still disciplined.
Color now carries emotion, not just decoration.

### Foundation colors

- Ivory Paper: `#f6efe8`
- Warm Mist: `#efe5df`
- Moon White: `#f7f6fb`
- Ink Brown: `#201c1d`
- Soft Ink: `#554d52`
- Border: `rgba(33, 27, 35, 0.14)`

### Atmosphere colors

- Twilight Indigo: `#5f6f97`
- Rain Blue: `#7d91bb`
- Piano Night: `#1d1e2c`
- Sakura Glow: `#dcb6c6`
- Evening Gold: `#d2b082`
- Haze Lavender: `#c7c4dc`

### Rules

- base layers remain warm and readable
- indigo and rain blue create depth and evening air
- sakura glow is used as a light bloom, not a theme color everywhere
- piano night is for depth, contrast, and emotional weight
- gold appears only as a rare warm spark

## 6. Typography

### Typography concept

The typography should now feel both literary and musical.

Headlines should feel like they arrive with air around them.
Body text should remain clear and modern.
Small labels should feel like track names, scene markers, or quiet metadata.

### Suggested font roles

- Headings: `Cormorant Garamond`, `Noto Serif JP`
- Body: `Inter`, `Noto Sans JP`
- Poetic or Japanese fragments: `Noto Serif JP`

### Rules

- headings can be slightly more dramatic than before
- body text must remain readable and grounded
- metadata can feel a little more cinematic and spaced out
- do not use harsh bold weights everywhere
- let rhythm and line breaks carry emotion

## 7. Layout Principles

### Core rules

- preserve breathing room
- add layered depth, not clutter
- let each section feel like a scene
- mix clarity with atmosphere
- use asymmetry to create emotional movement

### Density

The site should no longer feel sparse in a dry way.
It should feel full of air, light, and subtle texture.

### Containers

- page max-width around 1120px
- text width around 680px
- cards can feel like illuminated glass-paper panels
- sections should have distinct tonal shifts, like scenes changing in a film

## 8. Component Language

### Navigation

- translucent and elegant
- slightly more luminous than before
- feels like glass catching evening light
- still restrained, never flashy

### Hero

The hero should feel like the emotional thesis of the site.

It should include:

- identity
- atmospheric statement
- emotional weather
- visual hint of music / sky / distance

### Cards

Cards should no longer feel only like notes on paper.
They should feel like illuminated memory fragments.

- soft gradient depth
- subtle glass / mist layering
- thin borders
- richer glow around edges or highlights
- hover should feel like a gentle lift in air

### Links and buttons

- slightly more luminous
- can use soft gradient or dusk-toned fills
- interactions should feel precise and floating, not mechanical

## 9. Motion Language

Animation should feel like suspended air, rain drift, or resonance.

Allowed:

- slow parallax-like background drift if subtle
- fade and glow transitions
- tiny floating movement
- ambient shimmer at very low intensity
- gentle vertical motion

Avoid:

- flashy sparkle effects
- fast animated gradients
- bouncing or arcade-like movement
- attention-seeking motion loops

## 10. Imagery Direction

If imagery is introduced later, prefer:

- evening skies
- train lines, windows, rooftops
- rain on glass
- light through curtains
- piano keys, sheet music, silhouettes, reflections
- empty streets, distant buildings, clouds, summer air

Avoid:

- stock photography
- obvious fandom collages
- literal anime character art
- generic developer illustrations

## 11. Content Voice

The writing voice should feel:

- precise
- emotionally aware
- reflective
- intelligent
- lightly lyrical
- intimate without oversharing

This site should read like a place where technical thought and private atmosphere can coexist.

## 12. Current Implementation Target

For this second visual pass, the homepage should communicate:

- more emotional richness
- visible atmosphere in the background and panels
- a stronger cinematic first impression
- better use of indigo, blush, glow, haze, and night depth
- a sense that this site belongs to someone drawn to music, memory, weather, and distance

Without ever saying that directly.

## 13. Practical Do / Don't

### Do

- add depth through layered color and light
- use dusk tones and luminous accents carefully
- make the page feel like a scene, not just a layout
- allow emotional richness without losing discipline
- preserve readability while increasing atmosphere

### Don't

- become loud or over-illustrated
- turn the homepage into fandom display
- overuse Japanese text as ornament
- drown the page in gradients with no structure
- abandon the personal essay identity

## 14. Working Summary

The visual direction is now:

- Japanese-inspired
- cinematic
- music-shaped
- twilight-toned
- rain-softened
- luminous and melancholic
- elegant, but no longer plain
