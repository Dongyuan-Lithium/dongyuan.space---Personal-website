# Dongyuan.space Style Reference v1

## 1. Design Direction

This site should feel calm, literate, and quietly beautiful.

The core direction is not neon Tokyo, anime, or decorative "Japanese" symbolism. It is a softer and more restrained interpretation:

- Japanese editorial minimalism
- quiet luxury rather than loud branding
- paper-like warmth instead of glossy tech gradients
- emotional clarity instead of feature overload
- personal writing space rather than startup landing page

Working phrase:

"A quiet digital room for thoughts, code, and life."

## 2. Mood Keywords

Primary mood:

- quiet
- refined
- breathable
- thoughtful
- poetic
- natural
- restrained
- intimate

Secondary mood:

- modern
- literate
- tactile
- cinematic in a soft way
- reflective

Avoid:

- loud gradients
- overly corporate SaaS design
- flashy animations
- glassmorphism-heavy UI
- gamer / cyberpunk aesthetic
- cluttered card grids
- too many accent colors

## 3. Visual Identity

### 3.1 Overall feeling

The homepage should feel like a well-designed independent journal.
It should look personal and intentional, with enough polish to feel contemporary, but never too commercial.

### 3.2 Design references

Closest families of influence:

- Japanese editorial websites
- premium stationery / print magazines
- MUJI-like restraint
- Apple-like whitespace discipline
- Notion-like warm neutrality

This site should combine:

- Apple's spacing discipline
- Notion's paper warmth
- a Japanese editorial sense of emptiness and pause

## 4. Color System

The palette should remain soft and limited.
Most of the site should be neutral, with color used as punctuation instead of decoration.

### Primary palette

- Paper: `#f6f1ea`
- Paper Deep: `#efe7dc`
- Ink: `#1f1b18`
- Ink Soft: `#5f5954`
- Border: `rgba(31, 27, 24, 0.12)`
- Mist: `#d9d2c7`

### Accent palette

Use accents sparingly.

- Moss: `#7b8b7a`
- Sakura Dust: `#d7b7b0`
- Indigo Ash: `#6b7487`
- Gold Sand: `#b89b72`

Rule:

- neutrals dominate
- moss is the primary UI accent
- sakura dust is used only for soft highlights or hover warmth
- never use all accents loudly in one screen

## 5. Typography

### Typography concept

The type system should feel editorial.
The heading voice should be elegant and slightly literary.
The body should be modern, readable, and calm.

### Suggested font roles

- Headings: `Cormorant Garamond`, `Noto Serif JP`, serif fallback
- Body: `Inter`, `Noto Sans JP`, system sans-serif fallback
- Micro labels / metadata: `Inter`, `Noto Sans JP`

### Rules

- headings should feel graceful, never bulky
- body text should feel contemporary and clean
- metadata can use small uppercase or letter-spaced caps
- avoid very heavy font weights
- use line height generously for body copy

## 6. Layout Principles

### Core rules

- generous whitespace
- clear section rhythm
- readable content width
- asymmetry used carefully
- one strong focal element per section

### Density

This site should feel spacious.
Do not try to fill every area.
Empty space is part of the design language.

### Containers

- page content max-width: around 1120px
- reading width for text blocks: around 680px
- use horizontal breathing room generously on desktop
- mobile should still feel airy, not compressed

## 7. Component Language

### Navigation

- light, unobtrusive, almost editorial
- transparent or paper-toned background
- thin border lines
- no oversized CTA buttons in nav

### Hero

The hero should communicate identity, not marketing.

Preferred ingredients:

- name
- one-line quiet statement
- one slightly longer intro paragraph
- subtle bilingual or Japanese microcopy is allowed, but should remain tasteful and minimal

### Cards

Cards should feel like printed notes placed on a desk.

- soft background difference
- thin border
- low shadow or no shadow
- rounded corners, but not overly soft
- hover movement should be subtle

### Links and buttons

- minimal chrome
- underlines, thin borders, soft fills
- motion should be quiet and precise

## 8. Motion Language

Animation should feel like breath, not performance.

Allowed:

- soft fade-in
- tiny translate on hover
- low-contrast transitions
- slow background drift if almost imperceptible

Avoid:

- bounce
- spin
- shiny sweeps
- exaggerated parallax
- aggressive scroll animation

## 9. Content Voice

The writing voice should feel:

- reflective
- observant
- precise
- warm but not sentimental
- intelligent without trying too hard

The site is a personal space for:

- essays
- notes on technology
- life reflections
- selected projects

It should not read like a startup homepage.

## 10. Photography / Imagery Direction

If imagery is used later, prefer:

- quiet landscapes
- architectural stillness
- shadows on walls
- paper, books, desks, rain, trees, windows
- muted natural tones

Avoid:

- generic stock business photos
- loud illustrations
- random tech mockups everywhere

## 11. First Implementation Scope

For v1, the homepage should include:

- refined hero section
- short philosophy / intro section
- selected writings section
- approach / themes section
- about section
- quiet footer

The homepage should already reflect the style system, even before a full blog system exists.

## 12. Future Extensions

When the site grows, the following pages should inherit the same system:

- writing archive
- project page
- about page
- contact page
- individual essay pages

The style system should remain consistent across them:

- same paper-toned base
- same serif/sans pairing
- same restrained accent logic
- same spacing rhythm
- same quiet motion rules

## 13. Practical Do / Don't

### Do

- keep the interface calm
- use whitespace intentionally
- let typography carry beauty
- keep color restrained
- make the site feel personal and authored
- prefer elegance over cleverness

### Don't

- turn it into a gradient-heavy portfolio template
- overuse Japanese characters decoratively
- over-animate the homepage
- use too many cards or too much UI chrome
- make it look like a SaaS landing page
- chase trendiness at the cost of personality

## 14. Current v1 Interpretation

For the first redesign pass, interpret "Japanese beauty" as:

- warm paper background
- dark ink typography
- serif-led hero
- moss green as the main accent
- thin borders and quiet panels
- editorial spacing
- soft atmospheric textures

This is the baseline.
Later revisions can push toward one of these branches if desired:

- more wabi-sabi
- more literary magazine
- more contemporary Tokyo design
- more minimal luxury
- more poetic personal journal
